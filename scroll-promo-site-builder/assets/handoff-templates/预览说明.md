# 本地预览说明

## 环境

安装 `package.json` 所要求版本的 Node.js。

## macOS

双击 `start-mac.command`，或在终端执行：

```bash
npm install
npm run dev
```

## Windows

双击 `start-windows.bat`，或在命令提示符执行：

```bat
npm install
npm run dev
```

浏览器访问终端显示的 Local URL，默认通常为 `http://localhost:5173`。

## 体验

- 桌面端向下滚动推进视频，向上滚动回退视频。
- 手机端展示静态场景故事。
- 按 `Control + C` 停止本地服务。
