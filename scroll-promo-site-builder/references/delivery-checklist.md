# Delivery checklist

## Product and content

- [ ] Positioning is understandable in five seconds.
- [ ] Every scene has one clear action and claim.
- [ ] Proof cases are real and accurately labeled in DOM text.
- [ ] Final scene shows a tangible deliverable.
- [ ] CTA links are correct.

## Model provenance

- [ ] Image assets were generated directly with image2.0.
- [ ] Video assets were generated directly with Seedance 2.0.
- [ ] Prompts and references are saved.
- [ ] Asset manifest is complete.

## Media

- [ ] All clips pass ffprobe inspection.
- [ ] All clips share compatible dimensions, fps, codec, and pixel format after normalization.
- [ ] Every boundary has a recorded strategy.
- [ ] Full review video decodes without errors.
- [ ] Web video is H.264, yuv420p, faststart, and short-GOP.

## Website

- [ ] Down-scroll advances video.
- [ ] Up-scroll reverses video.
- [ ] Video frames move continuously; the desktop is not an image carousel.
- [ ] Video takes over from poster after readiness.
- [ ] Navigation, copy, progress, and video timeline agree.
- [ ] Mobile static story is readable.
- [ ] Reduced-motion and video-error states are readable.
- [ ] Production build passes.

## Handoff

- [ ] Local startup succeeds.
- [ ] macOS and Windows startup scripts are included.
- [ ] Preview instructions are included.
- [ ] Archive excludes node_modules, caches, build artifacts, and version-control data.
- [ ] Archive integrity check passes.
- [ ] No public deployment or source push occurred without explicit authorization.
